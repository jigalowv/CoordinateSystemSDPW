# 2. Реалізація

Було реалізовано методи оновлення для трьох фільтрів.

## SMA

```python
def update(self, x):
    if len(self.q) == self.w:
        self.sum -= self.q[0]
    self.q.append(x)
    self.sum += x
    return self.sum / len(self.q)
```

## EMA

```python
def update(self, x):
    if self.last is None:
        self.last = x
    else:
        self.last = self.a * x + (1 - self.a) * self.last
    return self.last
```

## Median

```python
def update(self, x):
    self.q.append(x)
    return np.median(self.q)
```
